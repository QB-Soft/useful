  • generateNumberInRange(lowerLimit?: number, upperLimit?: number, returnFloat?: boolean): number

    Returns a number between a lower and upper limit non-inclusive. Default limit is 0 - 100 if no parameters are specified. Can also return a float to 2 decimal places if required.

     generateNumberInRange() → 54
 generateNumberInRange(100, 200) → 145
 generateNumberInRange(-200, -100, true) → -145.34
